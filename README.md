# House Price Prediction

## Project Overview
Built a Machine Learning model to predict house prices based on property features and market data.

## Key Features
- Cleaned and prepared housing data for predictive modeling.
- Performed exploratory data analysis to identify key price drivers.
- Built regression models to estimate house prices.
- Evaluated model performance using MAE and RMSE.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Performance Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
